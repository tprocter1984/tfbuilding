This plugin will replace regular expressions in the final rendered WordPress output.


== Description ==

WP RegEx Replace gives an interface through which we can replace regular expressions on our WordPress rendered output.

It is very useful in scenarios like offloading static urls to CDN urls 

== Installation ==

To install WP RegEx Replace

1. Upload `wp-regex-replace.php` to the `/wp-content/plugins/wp-regex-replace/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How to assign WP Search and replace Expressions? =

Go to wp-admin >> Settings >> WP RegEx Replace and assign WP RegEx replacement expressions.
